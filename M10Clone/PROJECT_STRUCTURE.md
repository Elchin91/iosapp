# M10 Clone - Структура проекта

## Обзор
Клон приложения M10 Digital Wallet с AI-ассистентом, разработанный на SwiftUI для iOS 16+.

## Структура файлов

```
M10Clone/
├── M10Clone/
│   ├── M10CloneApp.swift              # Точка входа приложения
│   ├── ContentView.swift               # Главный TabView контейнер
│   ├── Info.plist                      # Конфигурация приложения
│   │
│   ├── Views/                          # UI компоненты
│   │   ├── AIChatView.swift           # AI чат интерфейс (главная фича)
│   │   ├── HomeView.swift             # Главная страница с балансом
│   │   ├── PaymentsView.swift         # Платежи и услуги
│   │   ├── TransfersView.swift        # Переводы денег
│   │   └── ProfileView.swift          # Профиль пользователя
│   │
│   ├── ViewModels/                     # Бизнес-логика
│   │   └── AIChatViewModel.swift      # ViewModel для AI чата
│   │
│   ├── Models/                         # Модели данных
│   │   └── Message.swift              # Модель сообщения чата
│   │
│   └── Extensions/                     # Расширения и утилиты
│       └── Colors.swift               # Цветовая палитра M10
│
└── PROJECT_STRUCTURE.md               # Этот файл
```

## Основные компоненты

### 1. AI Chat (Главная функция)
**Файл:** `Views/AIChatView.swift`
- Полноэкранный чат интерфейс
- Пузырьки сообщений (user справа, AI слева)
- Поле ввода с кнопками "прикрепить" и "отправить"
- Индикатор печати AI
- Автоматическая прокрутка к новым сообщениям

**ViewModel:** `ViewModels/AIChatViewModel.swift`
- Управление состоянием сообщений
- Логика отправки сообщений
- Генерация AI ответов (симуляция)

### 2. Home View
**Файл:** `Views/HomeView.swift`
- Карточка баланса с градиентом
- Быстрые действия (отправить, получить, обменять, пополнить)
- Список последних транзакций

### 3. Payments View
**Файл:** `Views/PaymentsView.swift`
- Поиск услуг
- Сетка категорий платежей
- Недавние платежи

### 4. Transfers View
**Файл:** `Views/TransfersView.swift`
- Выбор типа перевода (телефон/карта/счет)
- Недавние контакты
- Шаблоны переводов
- Информационный баннер

### 5. Profile View
**Файл:** `Views/ProfileView.swift`
- Профиль пользователя
- Список карт
- Настройки (основные, поддержка, приложение)
- Кнопка выхода

## Технологический стек

- **Фреймворк:** SwiftUI
- **Минимальная iOS:** 16.0+
- **Архитектура:** MVVM
- **Паттерны:**
  - Observable Objects (@StateObject, @Published)
  - Async/await для асинхронных операций
  - Protocol-oriented design

## Функциональность AI Chat

AI ассистент умеет отвечать на вопросы о:
- Балансе и транзакциях
- Платежах и оплате услуг
- Переводах денег
- Общей помощи с приложением

**Примеры запросов:**
- "Какой у меня баланс?"
- "Как сделать платеж?"
- "Помоги с переводом"
- "Помощь"

## Запуск проекта

1. Откройте Xcode
2. File → New → Project → iOS → App
3. Скопируйте все файлы из папки M10Clone в новый проект
4. Убедитесь, что deployment target установлен на iOS 16.0+
5. Запустите проект на симуляторе или устройстве
