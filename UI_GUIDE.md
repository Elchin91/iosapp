## UI Implementation Tips

### MessageBubble Alignment
```swift
HStack {
    if message.isUser {
        Spacer(minLength: 60)  // Left space for user
    }

    // Bubble content

    if !message.isUser {
        Spacer(minLength: 60)  // Right space for AI
    }
}
```

### Gradient Backgrounds
```swift
LinearGradient(
    colors: [AppColors.primary, AppColors.primaryDark],
    startPoint: .topLeading,
    endPoint: .bottomTrailing
)
```

### Conditional Button State
```swift
.foregroundColor(
    messageText.isEmpty ?
    AppColors.textSecondary :
    AppColors.primary
)
```

### Auto-scroll Pattern
```swift
ScrollViewReader { proxy in
    ScrollView {
        // Content
    }
    .onChange(of: messages.count) { _, _ in
        withAnimation {
            proxy.scrollTo(lastMessage.id, anchor: .bottom)
        }
    }
}
```
